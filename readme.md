# 睡眠盒子 (Sleep Box)

让白噪声伴你入眠 - 一个简洁、易用的白噪声助眠网站

## 📋 项目介绍

睡眠盒子是一个在线白噪声助眠工具，提供多种环境音效和预设场景，帮助用户放松身心、改善睡眠质量。

## ✨ 功能特性

### 🎵 丰富的音效库
- **自然类**：雨声、森林声、篝火声
- **海边类**：海水声、海浪声、海鸥声
- **禅意类**：颂钵声、诵经声、古筝声、钢琴声
- **基础噪声**：白噪声、粉噪声、棕噪声

### 🎯 预设场景
- 雨天森林：雨声 + 森林声
- 篝火营地：篝火声 + 森林声
- 海边夜晚：海水声 + 海浪声 + 海鸥声
- 古寺禅意：诵经声 + 颂钵声
- 琴茶一室：古筝声 + 钢琴声

### 🎛️ 灵活的控制
- 每种音效独立音量调节
- 总开关控制
- 平滑的淡入淡出效果
- 定时关闭功能（15/30/60/120分钟）

### 📱 响应式设计
- 适配桌面端和移动端
- 简洁美观的界面

### 🔒 隐私保护
- 无账号、无追踪、无广告
- 不收集任何用户数据

## 🚀 快速开始

1. **克隆项目**

## 🎮 使用指南

### 1. 选择预设场景
- 点击"预设场景"下的任意场景，系统会自动加载并播放该场景的音效组合

### 2. 自定义场景
- 切换到"自定义场景"标签页
- 选择音效类别（自然、海边、禅意、基础噪声）
- 点击每个音效的"播放"按钮，然后拖动滑块调整音量
- 可以同时播放多种音效，创造属于自己的独特环境

### 3. 设置定时
- 在页面底部选择定时时间（15/30/60/120分钟）
- 到达设定时间后，系统会自动停止播放
- 点击"关闭定时"可取消当前定时

### 4. 总控制
- 点击页面顶部的"开始播放"/"暂停播放"按钮，控制所有音效的播放状态

## 🔧 核心功能实现

### 音频引擎
- 使用Web Audio API创建音频上下文
- 实现多种噪声生成算法（白噪声、粉噪声、棕噪声）
- 支持音频文件播放和音量控制
- 平滑的淡入淡出效果

### 预设场景管理
- 预定义多种常见场景组合
- 一键切换不同场景
- 自动调整各音效音量

### 定时功能
- 支持多种定时选项
- 实时显示剩余时间
- 到达时间后自动停止播放
